# 冒烟测试执行检查清单

## 测试前准备

- [ ] MySQL 服务运行中
- [ ] Redis 服务运行中
- [ ] 后端服务已启动 (http://localhost:8080)
- [ ] 前端服务已启动 (http://localhost:5173)
- [ ] Playwright 已安装

## 快速运行

**Windows PowerShell (一键启动)**
```powershell
.\start-smoke-test.ps1
```

**手动运行**
```bash
# 1. 启动后端
cd backend
go run main.go

# 2. 启动前端 (新终端)
cd frontend
npm run dev

# 3. 运行测试 (新终端)
.\frontend\tests\run-smoke-test.ps1
```
