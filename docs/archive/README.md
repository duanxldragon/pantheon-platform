# Archive Directory

This directory contains historical documents and checklists that are kept for reference purposes but are no longer actively maintained.

## Contents

- `SYSTEM_CHECKLIST_2026-03-24.md` - Historical system audit checklist from March 2026
- `BACKEND_RELEASE_2026-04-01.md` - Historical release migration guide for April 2026 release

## Note

Documents in this directory are kept for historical reference only. They may not reflect the current state of the system. For current documentation, please refer to the main `docs/` directory.

## Archive Criteria

Documents are moved to this directory when they meet one of the following criteria:

1. **Completed execution plans** - Documents that outlined specific tasks or plans that have been completed
2. **Historical audits** - Checklists and audits that document the state of the system at a specific point in time
3. **Previous releases** - Release notes, migration guides, or runbooks for previous releases
4. **Superseded documents** - Documents that have been replaced by newer versions

## Maintenance

This directory is not actively maintained. Documents may contain outdated information or references to deprecated features.
