# Pantheon Platform 文档入口

> `docs/INDEX.md` 保留为兼容入口。
> 当前规范化文档导航统一以 `docs/DOCS_INDEX.md` 为准。

## 建议入口

- 平台文档总索引：`docs/DOCS_INDEX.md`
- 系统架构设计：`docs/design/SYSTEM_ARCHITECTURE.md`
- 开发流程入口：`docs/development/DEVELOPER_GUIDE.md`
- 工程开发流程：`docs/governance/ENGINEERING_WORKFLOW.md`
- 功能交付模板：`docs/development/FEATURE_DELIVERY_TEMPLATE.md`
- Code Review 规范：`docs/governance/CODE_REVIEW_GUIDE.md`
- 测试策略：`docs/testing/TEST_STRATEGY.md`

## 规范类文档主路径

- 平台级 UI 设计：`docs/system/UI_DESIGN.md`
- 前端代码规范：`frontend/docs/FRONTEND_CODE_STANDARDS.md`
- 后端代码规范：`backend/docs/BACKEND_CODE_STANDARDS.md`
- 编码与换行：`docs/governance/ENCODING_AND_LINE_ENDINGS.md`

## 说明

- `docs/DOCS_INDEX.md` 是平台级主入口；
- `backend/docs/` 与 `frontend/docs/` 分别承接实现视角文档；
- 根目录 `docs/` 下的大量阶段性报告与评估文档默认视为过程材料，不作为开发规范主入口。
