## 🎯 快速开始

### 系统登录

#### 访问系统
1. 打开浏览器，访问系统地址
2. 进入登录页面
3. 输入用户名和密码
4. 点击"登录"按钮

#### 首次登录
1. 首次登录后，系统会提示修改密码
2. 新密码必须符合安全要求：
   - 至少12个字符
   - 包含大小写字母
   - 包含数字和特殊字符
   - 不包含用户名或常见单词

#### 忘记密码
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收邮件，点击重置链接
4. 设置新密码

## 👤 用户管理

### 用户列表

#### 查看用户
1. 点击侧边栏"用户管理"
2. 查看"用户列表"页面
3. 显示所有用户信息：
   - 用户名
   - 真实姓名
   - 邮箱
   - 手机号
   - 所属部门
   - 职位
   - 状态

#### 搜索用户
1. 在用户列表页面，使用搜索框
2. 输入用户名、姓名或邮箱
3. 点击搜索按钮或按回车键
4. 系统显示匹配结果

#### 高级筛选
1. 点击"高级筛选"按钮
2. 设置筛选条件：
   - 用户状态（正常/禁用）
   - 所属部门
   - 职位
   - 创建时间范围
3. 点击"应用筛选"

### 创建用户

#### 单个创建
1. 在用户列表页面，点击"新建用户"按钮
2. 填写用户信息：
   - 用户名（必填，唯一）
   - 真实姓名（必填）
   - 邮箱（必填，唯一）
   - 手机号
   - 所属部门
   - 职位
   - 分配角色
   - 账户状态
3. 点击"保存"按钮

#### 批量导入
1. 点击"批量导入"按钮
2. 下载导入模板
3. 按模板格式填写用户信息
4. 上传填写好的Excel文件
5. 系统验证数据格式
6. 确认导入信息
7. 开始导入
8. 查看导入结果

### 编辑用户

#### 修改用户信息
1. 在用户列表中，找到要编辑的用户
2. 点击操作列的"编辑"按钮
3. 修改用户信息
4. 点击"保存"按钮

#### 重置用户密码
1. 在用户列表中，找到目标用户
2. 点击操作列的"更多" > "重置密码"
3. 系统生成临时密码
4. 将临时密码发送给用户
5. 用户首次登录需要修改密码

#### 启用/禁用用户
1. 在用户列表中，找到目标用户
2. 点击操作列的"启用"或"禁用"按钮
3. 确认操作
4. 用户状态立即生效

### 删除用户

1. 在用户列表中，找到要删除的用户
2. 点击操作列的"删除"按钮
3. 系统弹出确认对话框
4. 确认删除操作
5. 用户数据将被软删除（保留在数据库中）

## 🔐 角色权限管理

### 角色管理

#### 查看角色列表
1. 点击侧边栏"角色管理"
2. 系统显示所有角色：
   - 角色名称
   - 角色描述
   - 用户数量
   - 创建时间
   - 状态

#### 创建角色
1. 点击"新建角色"按钮
2. 填写角色信息：
   - 角色名称（必填）
   - 角色描述
   - 角色状态
3. 配置权限：
   - 菜单权限
   - 操作权限
   - 数据权限
4. 点击"保存"按钮

#### 编辑角色权限
1. 在角色列表中，找到要编辑的角色
2. 点击操作列的"编辑权限"按钮
3. 系统显示权限配置界面：
   - **菜单权限**: 勾选角色可访问的菜单
   - **按钮权限**: 勾选角色可执行的操作
   - **数据权限**: 设置数据访问范围
4. 修改权限配置
5. 点击"保存"按钮

#### 权限继承
- 角色可以继承其他角色的权限
- 在创建角色时，选择要继承的父角色
- 子角色自动获得父角色的所有权限
- 子角色可以扩展父角色的权限

### 权限分配

#### 为用户分配角色
1. 在用户编辑页面
2. 找到"角色分配"区域
3. 从可用角色列表中选择角色
4. 点击"添加"按钮
5. 角色添加到"已分配角色"列表
6. 保存用户信息

#### 查看用户权限
1. 在用户列表中，点击用户名
2. 进入用户详情页面
3. 查看"权限信息"标签
4. 显示用户的所有权限：
   - 用户所属角色
   - 角色对应的权限
   - 直接分配的权限

## 🏢 租户管理

### 租户概览

#### 租户列表
1. 点击侧边栏"租户管理"
2. 系统显示所有租户：
   - 租户名称
   - 租户代码
   - 状态（正常/暂停/冻结）
   - 用户数量
   - 存储使用情况
   - 创建时间

#### 租户健康状态
- 🟢 健康：系统运行正常
- 🟡 警告：资源使用超过阈值
- 🔴 危险：系统异常或资源耗尽

### 租户配置

#### 创建租户
1. 点击"新建租户"按钮
2. 填写租户基本信息：
   - 租户名称（必填）
   - 租户代码（必填，唯一）
   - 联系人姓名
   - 联系人邮箱
   - 联系人电话
   - 租户描述
3. 配置资源配额：
   - 最大用户数
   - 存储空间（GB）
   - API调用频率（次/分钟）
4. 配置数据库：
   - 数据库类型
   - 数据库版本
   - 连接池大小
5. 点击"创建"按钮

#### 编辑租户信息
1. 在租户列表中，找到要编辑的租户
2. 点击操作列的"编辑"按钮
3. 修改租户信息
4. 点击"保存"按钮

#### 租户状态管理
**正常** → **暂停**:
1. 选择要暂停的租户
2. 点击"暂停"按钮
3. 系统暂停租户服务
4. 用户无法登录
5. 数据保留

**暂停** → **正常**:
1. 选择要恢复的租户
2. 点击"恢复"按钮
3. 系统恢复租户服务
4. 用户可以正常登录

**正常** → **冻结**:
1. 选择要冻结的租户
2. 点击"冻结"按钮
3. 系统冻结租户账户
4. 准备删除数据

**冻结** → **删除**:
1. 冻结30天后可以删除
2. 点击"删除"按钮
3. 系统永久删除租户数据
4. 此操作不可逆

## ❓ 常见问题

### 登录相关

**Q: 忘记登录密码怎么办？**
A: 点击登录页面的"忘记密码"链接，输入注册邮箱，系统会发送密码重置链接到邮箱。

**Q: 账户被锁定怎么办？**
A: 多次输入错误密码会导致账户锁定，请联系管理员解锁，或等待30分钟自动解锁。

**Q: 为什么我无法登录系统？**
A: 请检查以下几点：
1. 用户名和密码是否正确
2. 账户状态是否正常（未禁用）
3. 浏览器是否支持（推荐Chrome、Firefox、Edge）
4. 网络连接是否正常

### 权限相关

**Q: 为什么我看不到某个菜单？**
A: 这是因为当前用户角色没有该菜单的访问权限，请联系管理员分配相应权限。

**Q: 为什么有些按钮是灰色的？**
A: 灰色按钮表示当前用户没有该操作的权限，需要联系管理员授权。

**Q: 如何申请更多权限？**
A: 请联系您的系统管理员，说明需要的权限和原因，管理员会进行审核和分配。

### 数据相关

**Q: 如何批量导入用户数据？**
A: 在用户列表页面，点击"批量导入"按钮，下载Excel模板，按模板格式填写数据后上传。

**Q: 导入数据失败怎么办？**
A: 请检查：
1. 数据格式是否正确
2. 必填字段是否完整
3. 数据是否重复（如用户名、邮箱不能重复）
4. 查看导入失败的具体错误信息

**Q: 如何导出数据？**
A: 在列表页面，点击"导出"按钮，选择导出格式（Excel、CSV、PDF），系统会生成文件供下载。

### 性能相关

**Q: 系统响应速度慢怎么办？**
A: 可能的原因和解决方法：
1. 网络问题：检查网络连接
2. 浏览器问题：清除缓存和Cookie
3. 系统负载高：避开高峰期使用
4. 数据量大：使用筛选条件缩小查询范围

**Q: 如何提高系统使用效率？**
A: 建议的做法：
1. 使用收藏夹功能，快速访问常用页面
2. 合理使用筛选功能，快速定位数据
3. 使用快捷键，提高操作效率
4. 定期清理浏览器缓存
