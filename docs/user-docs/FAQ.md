## 🔐 账户与登录

### Q1: 忘记密码怎么办？

**问题描述**: 我忘记了登录密码，无法进入系统。

**解决方案**:
1. 在登录页面点击"忘记密码"链接
2. 输入注册时使用的邮箱地址
3. 检查邮箱，找到密码重置邮件
4. 点击邮件中的重置链接
5. 输入新密码（需符合密码复杂度要求）
6. 使用新密码登录

**注意事项**:
- 密码重置链接30分钟内有效
- 如果没有收到邮件，检查垃圾邮件文件夹
- 如仍无法解决，联系管理员重置密码

---

### Q2: 账户被锁定如何处理？

**问题描述**: 登录时提示"账户已被锁定"。

**可能原因**:
- 连续5次输错密码
- 管理员手动锁定
- 安全策略触发

**解决方案**:
1. **自动锁定**: 等待30分钟后自动解锁
2. **联系管理员**: 如果需要立即解锁，联系系统管理员
3. **密码重置**: 解锁后建议重置密码

**预防措施**:
- 使用密码管理器记住密码
- 避免多次输错密码
- 定期更换密码

---

### Q3: 首次登录系统提示修改密码？

**问题描述**: 管理员创建了账户，首次登录时强制修改密码。

**操作步骤**:
1. 使用初始密码登录
2. 系统自动跳转到密码修改页面
3. 输入当前密码（初始密码）
4. 输入新密码（符合复杂度要求）
5. 确认新密码
6. 点击"提交"完成修改

**密码复杂度要求**:
- 最少12个字符
- 包含大写字母
- 包含小写字母
- 包含数字
- 包含特殊字符
- 不能与用户名相同
- 不能与最近3次密码相同

---

### Q4: 为什么会自动退出登录？

**问题描述**: 使用一段时间后，系统自动退出到登录页面。

**可能原因**:
- 会话超时（默认30分钟无操作）
- 令牌过期（访问令牌2小时，刷新令牌7天）
- 系统更新或重启
- 账户在另一设备登录

**解决方案**:
- **会话超时**: 重新登录即可
- **令牌过期**: 系统会自动刷新，如失败则重新登录
- **系统更新**: 等待更新完成后重新登录
- **异地登录**: 检查是否为本人操作，注意账户安全

## 👥 用户管理

### Q5: 如何批量导入用户？

**问题描述**: 需要一次性创建大量用户账户。

**操作步骤**:
1. 进入"用户管理" → "用户列表"
2. 点击"批量导入"按钮
3. 下载Excel模板文件
4. 按模板格式填写用户信息：
   ```
   用户名* | 真实姓名* | 邮箱* | 手机号 | 部门 | 职位 | 角色
   john.doe | 约翰·多伊 | john@example.com | 138... | 技术部 | 工程师 | user
   ```
5. 上传填写好的Excel文件
6. 系统自动验证数据格式
7. 预览导入信息，确认无误后开始导入
8. 查看导入结果统计

**注意事项**:
- 必填字段不能为空
- 用户名和邮箱必须唯一
- 角色必须在系统中已存在
- 导入失败记录会在结果中标注原因
- 建议先导入少量数据测试

---

### Q6: 如何禁用用户账户？

**问题描述**: 员工离职或需要暂停某个用户的使用权限。

**操作步骤**:
1. 进入"用户管理" → "用户列表"
2. 搜索找到要禁用的用户
3. 点击该用户行的"禁用"按钮
4. 选择禁用原因：
   - 违反公司规定
   - 账户安全风险
   - 离职/转岗
   - 其他原因
5. 添加备注说明（可选）
6. 确认禁用

**禁用效果**:
- 用户立即无法登录系统
- 已登录的会话会被强制退出
- 用户数据保留，可以重新启用

---

### Q7: 用户信息导出后如何打开？

**问题描述**: 导出的用户数据文件无法正常打开或显示乱码。

**解决方案**:

**Excel文件**:
1. 使用Microsoft Excel 2016或更高版本
2. 使用WPS Office
3. 如显示乱码，使用"导入数据"功能，选择UTF-8编码

**CSV文件**:
1. 使用文本编辑器（记事本、VS Code）打开
2. Excel打开时选择"数据" → "从文本/CSV导入"
3. 选择UTF-8编码

**PDF文件**:
1. 使用Adobe Acrobat Reader
2. 或其他PDF阅读器（如浏览器直接打开）

## 🔑 角色与权限

### Q8: 如何创建自定义角色？

**问题描述**: 需要创建一个新角色并分配特定权限。

**操作步骤**:
1. 进入"角色管理"页面
2. 点击"新建角色"按钮
3. 填写角色基本信息：
   ```
   角色名称: 数据分析师
   角色代码: data_analyst
   角色描述: 负责数据分析和报表生成
   是否系统角色: 否
   ```
4. 点击"创建"按钮
5. 在角色列表中找到新创建的角色
6. 点击"编辑权限"按钮

**配置权限**:
1. **菜单权限**: 在左侧菜单树中勾选可访问的菜单
2. **操作权限**: 配置增删改查权限
   - 查看权限
   - 新增权限
   - 编辑权限
   - 删除权限
   - 导出权限
3. **数据权限**: 选择数据范围
   - 全部数据
   - 本部门数据
   - 仅本人数据
4. 点击"保存"完成配置

---

### Q9: 为什么用户看不到某个菜单？

**问题描述**: 分配了角色，但用户仍然看不到某些菜单。

**可能原因**:
1. **菜单权限未分配**: 角色没有被授予该菜单的访问权限
2. **角色未分配**: 用户没有被分配该角色
3. **菜单已禁用**: 菜单在系统配置中被禁用
4. **父级菜单未授权**: 菜单的父级菜单权限未授予
5. **缓存未更新**: 权限变更后缓存未刷新

**解决方案**:
1. 检查角色的菜单权限配置
2. 确认用户已分配该角色
3. 检查菜单状态是否为"启用"
4. 确保父级菜单权限已授予
5. 用户重新登录刷新权限缓存

---

### Q10: 权限变更后多久生效？

**问题描述**: 修改了用户权限，但用户反馈还是看不到变化。

**生效时间**:
- **立即生效**: 大部分权限变更立即生效
- **重新登录**: 某些权限需要用户重新登录
- **缓存刷新**: 系统每5分钟刷新一次权限缓存

**强制生效方法**:
1. 用户退出登录
2. 清除浏览器缓存
3. 重新登录系统

## 💻 系统操作

### Q11: 页面加载缓慢怎么办？

**问题描述**: 系统响应慢，页面加载时间长。

**可能原因**:
1. **网络问题**: 本地网络不稳定或速度慢
2. **浏览器问题**: 浏览器缓存过多或插件冲突
3. **服务器负载**: 系统访问量大，服务器压力大
4. **数据量**: 查询数据量过大

**解决方案**:

**用户端优化**:
1. 清除浏览器缓存
   - Chrome: Ctrl+Shift+Delete
   - Edge: Ctrl+Shift+Delete
   - Firefox: Ctrl+Shift+Delete
2. 禁用不必要的浏览器插件
3. 尝试使用其他浏览器
4. 检查网络连接
5. 关闭其他占用带宽的程序

**数据查询优化**:
1. 使用搜索功能精确查找
2. 设置合适的过滤条件
3. 减少单页显示的数据量
4. 避免在高峰期进行大量操作

**如果问题持续**:
- 联系管理员检查系统状态
- 查看系统公告是否有维护通知

### Q16: 导入失败如何处理？

**问题描述**: 批量导入数据时部分或全部失败。

**常见失败原因**:
1. **格式错误**: Excel格式不符合模板要求
2. **必填项缺失**: 必填字段为空
3. **数据重复**: 用户名或邮箱已存在
4. **格式不对**: 日期、数字格式不正确
5. **超出限制**: 数据量超过单次导入限制

**处理步骤**:
1. 查看导入结果详情
2. 下载失败记录文件
3. 根据失败原因修正数据
4. 重新导入修正后的数据

**预防措施**:
1. 使用最新模板
2. 检查必填字段
3. 检查数据格式
4. 先导入少量数据测试

## 🔒 安全问题

### Q19: 如何启用多因素认证 (MFA)？

**问题描述**: 提高账户安全性，启用MFA。

**操作步骤**:
1. 登录系统
2. 进入"个人设置" → "安全设置"
3. 点击"启用多因素认证"
4. 选择认证方式：
   - **认证器应用**（推荐）
     - 下载Google Authenticator或类似应用
     - 扫描QR码
     - 输入6位验证码确认
   - **短信验证**
     - 绑定手机号
     - 输入短信验证码确认
5. 保存备用恢复码（重要！）
6. MFA启用完成

**登录时使用MFA**:
1. 输入用户名和密码
2. 进入MFA验证页面
3. 打开认证器应用查看验证码
4. 输入6位验证码
5. 验证通过后登录成功

**备用恢复码**:
- 保存在安全的地方
- 每个恢复码只能使用一次
- 手机丢失时使用恢复码登录

### Q20: 发现可疑账户活动怎么办？

**问题描述**: 看到不认识的登录记录或异常操作。

**立即行动**:
1. **修改密码**: 立即修改账户密码
2. **检查活动**: 查看"登录历史"和"操作日志"
3. **清除会话**: 在"安全设置"中终止所有活跃会话
4. **启用MFA**: 如果尚未启用，立即启用多因素认证
5. **联系管理员**: 报告安全问题

**查看可疑活动**:
1. 进入"个人设置" → "安全设置"
2. 查看"登录历史"
   - 登录时间
   - 登录IP地址
   - 登录地点
   - 设备信息
3. 查看"操作日志"
   - 操作时间
   - 操作类型
   - 操作结果

**预防措施**:
- 使用强密码
- 启用MFA
- 不要在公共设备上保存密码
- 定期检查账户活动

## 🔔 通知问题

### Q26: 如何设置通知偏好？

**问题描述**: 想要控制接收哪些类型的通知。

**通知设置**:
1. 进入"个人设置" → "通知设置"
2. 配置通知渠道：
   - 系统内通知
   - 邮件通知
   - 短信通知（如支持）

3. 选择通知类型：
   - 系统通知
   - 审批通知
   - 提醒通知
   - 告警通知

4. 设置频率：
   - 实时通知
   - 每日汇总
   - 每周汇总
   - 不接收

---

### Q27: 没有收到通知邮件怎么办？

**问题描述**: 应该收到通知邮件但没有收到。

**可能原因**:
1. **邮件设置**: 关闭了邮件通知
2. **垃圾邮件**: 邮件被归类为垃圾邮件
3. **邮箱地址**: 邮箱地址错误或邮箱满了
4. **邮件服务器**: 邮件服务器延迟或故障
5. **邮箱配置**: 收件箱设置问题

**解决方案**:
1. 检查垃圾邮件文件夹
2. 检查通知设置是否开启
3. 验证邮箱地址是否正确
4. 检查邮箱空间是否已满
5. 添加系统邮箱到白名单
6. 联系管理员检查邮件服务器

## 🌐 多语言

### Q30: 如何切换界面语言？

**问题描述**: 需要使用英文或其他语言界面。

**操作步骤**:
1. 点击右上角用户头像
2. 选择"个人设置"
3. 在"界面设置"中选择语言：
   - 简体中文
   - English
4. 保存设置
5. 页面会自动刷新应用新语言

**注意事项**:
- 某些专业术语可能仍显示原文
- 用户输入的数据不会被翻译
